# User Stories — La Verde (copiar a GitHub Project)

## Must Have

- Como **visitante**, quiero ver el catálogo de productos con fotos y precios para elegir qué comprar.
- Como **usuario**, quiero registrarme e iniciar sesión con contraseña segura para acceder a mi cuenta.
- Como **usuario**, quiero agregar productos al carrito y modificar cantidades para armar mi pedido.
- Como **usuario**, quiero guardar favoritos para volver a comprarlos después.
- Como **usuario**, quiero confirmar el pedido y ver el historial en Mis pedidos.
- Como **usuario**, quiero recuperar mi contraseña si la olvidé.
- Como **admin**, quiero crear, editar y eliminar productos (CRUD) con imagen.
- Como **equipo**, necesitamos la app desplegada con URL pública.

## Should Have

- Como **admin**, quiero subir fotos con Cloudinary al crear productos.
- Como **usuario**, quiero buscar productos desde la barra del navbar.
- Como **usuario**, quiero filtrar por categoría en el catálogo.

## Could Have

- Como **usuario**, quiero pagar con Mercado Pago (sandbox).
- Como **admin**, quiero ver todos los pedidos de clientes.
- Como **usuario**, quiero recibir email real al recuperar contraseña.
